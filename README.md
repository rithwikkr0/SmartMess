🍽️ SmartMess Pro: Modern Campus Meal Management
SmartMess Pro is a comprehensive web-based solution designed to streamline college mess operations through digital meal booking, QR-based entry verification, and an integrated student wallet system.

🚀 Key Features
Unified Authentication: Secure login and signup flows for three distinct roles: Students, Staff, and Administrators.

Smart Wallet & Loyalty: Integrated digital wallet for instant meal booking and a loyalty points system to reward consistent users.

QR Entry Pass: Dynamic QR code generation for each booking to ensure contactless and rapid entry verification at the mess hall.

Live Dashboard: Real-time counters for wallet balance, total bookings, and time saved.

Admin Control Center: Tools for posting global announcements, viewing live booking logs, and downloading revenue reports.

Staff Scanner: A manual verification interface for staff to validate Booking IDs and grant access.

Interactive Feedback: A dedicated section for students to rate service quality and suggest improvements.

🛠️ Tech Stack
Frontend: HTML5, CSS3 (Custom Properties/Variables for Dark Mode support).

Logic: Vanilla JavaScript (ES6+) for state management and local storage persistence.

UI/UX: Responsive grid layouts and glassmorphism-inspired design.

API Integration: Utilizes the QRServer API for real-time QR code generation.
